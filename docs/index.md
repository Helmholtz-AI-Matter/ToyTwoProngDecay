# ToyTwoProngDecay docs

These docs are curated and notebook-driven: the published site is built from a
small set of marimo notebooks that act as the canonical source for examples,
plots, and narrative walkthroughs, while notebooks that are still exploratory,
experimental, or not yet polished enough for public consumption stay out of the
initial documentation build until they are ready.
